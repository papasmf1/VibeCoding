🏗️ HTML5

웹 페이지의 구조와 콘텐츠를 정의하는 마크업 언어입니다. HTML5는 시맨틱 태그와 멀티미디어 지원을 통해 더욱 강력해졌습니다.

  • 시맨틱 태그 (header, nav, article, section, footer)
  • 멀티미디어 지원 (audio, video)
  • Canvas를 통한 그래픽 처리
  • 웹 스토리지 (localStorage, sessionStorage)
  • 지오로케이션 API

🎨 CSS3

웹 페이지의 스타일과 레이아웃을 담당하는 스타일시트 언어입니다. CSS3는 애니메이션, 그라디언트, 플렉스박스 등 강력한 기능을 제공합니다.

  • 플렉스박스와 그리드 레이아웃
  • CSS 애니메이션과 트랜지션
  • 그라디언트와 그림자 효과
  • 반응형 디자인 (미디어 쿼리)
  • CSS 변수와 계산 함수

⚡ JavaScript

웹 페이지에 동적 기능과 상호작용을 추가하는 프로그래밍 언어입니다. ES6+의 현대적인 문법으로 더욱 강력해졌습니다.

  • DOM 조작과 이벤트 처리
  • 비동기 프로그래밍 (Promise, async/await)
  • 모듈 시스템 (import/export)
  • 화살표 함수와 구조 분해
  • 클래스와 상속
